# Correction: Enabling transparent research evaluation: A method for historical RCR retrieval using public NIH metadata

> **저자**: Fabian Haupt, J. Senge, Hendrik von Tengg-Kobligk, W. Bosbach | **날짜**: 2026 | **DOI**: [10.1371/journal.pone.0345661](https://doi.org/10.1371/journal.pone.0345661)

---

## Essence


본 논문은 원본 논문(DOI: 10.1371/journal.pone.0340697)의 데이터 가용성 명시(Data Availability statement)를 수정하는 정정(correction) 논문이다. 수정된 내용은 연구 원본 데이터가 figshare에 DOI: https://doi.org/10.6084/m9.figshare.30067852로 공개 접근 가능함을 명시한 것이다.

## Motivation

- **Known**: 원본 논문은 공개된 NIH 메타데이터를 활용하여 역사적 RCR(Research Compliance Record) 데이터 검색을 위한 방법론을 제시했다. 투명한 연구 평가를 가능하게 하기 위한 방법론이 제안되었다.
- **Gap**: 원본 논문의 데이터 가용성 명시가 불완전하였으므로, 연구의 재현성과 투명성을 보장하기 위해 명확한 데이터 접근 경로 제시가 필요했다.
- **Why**: 투명한 연구 평가와 과학적 재현성은 오픈 사이언스의 핵심 원칙이므로, 원본 데이터의 명확한 접근 정보 제공은 학술 공동체의 신뢰성 강화에 필수적이다.
- **Approach**: 원본 논문의 Data Availability statement를 수정하여 figshare 저장소의 정확한 DOI를 명시함으로써 연구 데이터에 대한 명확한 접근 경로를 제공한다.

## Achievement


- **데이터 가용성 명확화**: figshare 플랫폼에 저장된 원본 데이터의 정확한 DOI (https://doi.org/10.6084/m9.figshare.30067852)를 명시하여 접근성 개선
- **투명성 강화**: Data Availability statement 수정을 통해 오픈 액세스 원칙 준수 확인
- **재현성 보장**: 공개된 데이터 링크로 독립적인 검증 및 재현 연구 가능

## How


- 원본 논문(PLoS One 21(1):e0340697)의 Data Availability statement 내용 검토
- figshare 저장소의 정정된 DOI 확인 (https://doi.org/10.6084/m9.figshare.30067852)
- 공식 정정 논문 형식으로 PLoS One에 게재하여 원본 논문과의 연결 명시

## Limitation & Further Study

- 본 정정 논문은 원본 논문의 일부 정보 수정만을 다루므로 새로운 연구 내용이나 방법론 개선을 포함하지 않음
- Data Availability statement 수정만으로는 원본 논문의 다른 잠재적 오류를 수정하지 못함
- 후속 관련 연구에서는 figshare 데이터의 장기 보존성과 접근성 지속 여부를 모니터링할 필요
